<h2>📝 Project Description</h2>

<p>
<strong>Findora</strong> is a smart Lost & Found web application designed for colleges, schools, and campuses. 
It helps students and staff report lost or found items and automatically matches them using fuzzy search logic.
</p>

<p>
The platform integrates interactive location tracking using 
<a href="https://leafletjs.com/" target="_blank"><strong>Leaflet.js</strong></a> 
and intelligent item matching using 
<a href="https://fusejs.io/" target="_blank"><strong>Fuse.js</strong></a>, 
making the recovery process faster and more efficient.
</p>

<hr/>

<h2>🚀 Features</h2>

<ul>
  <li>🔐 User Authentication & Secure Login</li>
  <li>➕ Add Lost Items with title, description, image & location</li>
  <li>📦 Add Found Items with complete details</li>
  <li>🗺 Interactive Map Integration using Leaflet.js</li>
  <li>📍 Select exact location on map while reporting</li>
  <li>🔎 Smart Matching System using Fuse.js (Fuzzy Search)</li>
  <li>📸 Image Upload Support</li>
  <li>📂 Filter by Lost / Found</li>
  <li>📜 View Matched Results Instantly</li>
  <li>❌ Delete your posted item</li>
</ul>

<h2>🗺 How Matching Works</h2>

<p>
Findora uses <strong>Fuse.js</strong> to intelligently match items based on:
</p>

<ul>
  <li>Title similarity</li>
  <li>Description keywords</li>
  <li>Location name</li>
</ul>

<p>
This allows partial and typo-tolerant matching for better recovery chances.
</p>
